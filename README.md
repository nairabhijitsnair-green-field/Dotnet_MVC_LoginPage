# Dotnet MVC Login Page

A simple login and registration system built using .NET MVC and Session management.

## Features
- User Registration
- Login/Logout functionality
- Session-based state management
- Built with .NET Framework

## How to run locally
1. Clone the repository.
2. Open `Modified-Login.slnx` in Visual Studio.
3. Press `F5` to run the application.
